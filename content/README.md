# Content Surface

This directory is the authored game-content surface that is already committed in this repository.

Current live state:

- `content/characters/` is the only populated authored content tree at the moment.
- Other authored domains that may exist later are not being signaled as active subtrees until they contain real committed content.

This wording is intentionally narrow so the repository structure does not imply broader authored-content coverage than the current tree actually provides.
